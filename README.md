# Geoplot
GeoViz is a fast, client-side geophysical visualization toolkit. Easily plot VLF and Resistivity survey profiles, automatically calculate Apparent Resistivity (\rho_a = K \cdot R), and apply the Karous-Hjelt filter for quick analysis and interpretation. Supports multiple arrays (Wenner, Schlumberger, Dipole-Dipole). (338 characters)
